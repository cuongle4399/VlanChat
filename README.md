# VlanChat (LAN Chat Pro)

![Demo VlanChat](https://raw.githubusercontent.com/cuongle4399/cuongle4399/main/img/Screenshot%202026-05-24%20215743.png)

VlanChat là ứng dụng nhắn tin và truyền file nhanh chóng trong mạng nội bộ (LAN) được phát triển bằng C# WinForms (.NET).

## Chức năng chính
* **Chat nhóm & Chat riêng**: Hỗ trợ trò chuyện nhóm hoặc nhắn tin riêng tư giữa các thành viên.
* **Gửi file tốc độ cao**: Truyền tải file trực tiếp qua TCP giữa các máy khách, hiển thị tiến độ trực quan.
* **Quản lý trạng thái**: Xem danh sách thành viên đang online/offline.
* **Gửi Emoji**: Tích hợp các biểu tượng cảm xúc khi nhắn tin.
* **Cấu hình linh hoạt**: Thay đổi IP Server LAN dễ dàng trực tiếp trong phần Settings.
