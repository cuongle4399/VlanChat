# VlanChat (LAN Chat Pro)

![Demo VlanChat](https://raw.githubusercontent.com/cuongle4399/cuongle4399/main/img/Screenshot%202026-05-18%20204704.png)

VlanChat là ứng dụng nhắn tin và chia sẻ file trực tiếp trong mạng nội bộ (LAN) hoạt động theo mô hình Peer-to-Peer (P2P) hoàn toàn phi tập trung mà không cần máy chủ trung gian. Ứng dụng được phát triển trên nền tảng C# WinForms (.NET), hỗ trợ cơ chế tự động dò tìm các thiết bị trực tuyến qua giao thức UDP Discovery và truyền tải dữ liệu tốc độ cao qua kết nối TCP trực tiếp, đồng thời tối ưu hóa hiệu năng và dung lượng với hỗ trợ biên dịch Native AOT.
